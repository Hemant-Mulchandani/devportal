Aiven documentation
===================

Aiven is a database-as-a-service for open source data solutions including
PostgreSQL®,
M3,
Apache Kafka®,
Apache Flink®,
OpenSearch®,
Apache Cassandra®,
Redis®*,
MySQL,
InfluxDB® and Grafana®.
